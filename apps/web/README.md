# Web App (React Starter Skeleton)

Frontend web para administración/operación basado en React.

## Flujo inicial

- Login
- Listado de usuarios
- Perfil actual

## Scripts esperados

- `npm install`
- `npm run dev`
- `npm run test`
- `npm run e2e`

## API real

Crear `.env` con:

`VITE_API_BASE_URL=http://127.0.0.1:8000/api/v1`

## Checklist CI sugerido

- Unit: `npm test`
- E2E: `npm run e2e`
